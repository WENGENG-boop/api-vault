Local-first · Open source

Every AI API key, request, and token — in one local vault.

API Vault is an open-source, local-first proxy and dashboard. It encrypts your API keys on your own machine, forwards OpenAI- and Anthropic-compatible requests, and records every token, cost, and latency — without sending anything to the cloud.

Features

Built for control and privacy

Everything you need to manage AI API usage in one local tool.

Local-first security

Credentials are encrypted in a local vault file. Nothing is stored in the cloud.

Multi-provider proxy

Forwards OpenAI- and Anthropic-compatible requests to any upstream provider.

Token & latency audit

Records status, latency, and input/output tokens for every single call.

Balance sync

Custom HTTP polling and JSON-path rules sync provider-reported balances.

Cloudflared tunnels

Safely expose your local vault to remote apps through Cloudflare Tunnel.

Consolidated metrics

Groups keys by provider for both aggregate and per-key reports.

Quick start

Up and running in two steps

Choose Docker or the Windows start script. Your data stays on your machine.

Terminal 
# 1. Clone the repository
git clone https://github.com/WENGENG-boop/api-vault.git
cd api-vault

# 2. Build and start in the background
docker compose up -d --build

# 3. Open the dashboard
# http://localhost:3210
start-api-vault.bat 
# 1. Install Node.js LTS  ->  https://nodejs.org

# 2. Double-click start-api-vault.bat
#    It installs deps, builds, and launches the server.

# 3. Open the dashboard
# http://127.0.0.1:3210

Keep your keys. Keep your data.
